## StreamVibe - верстка статичного многостраничного сайта

Стэк технологий: [Minista](https://minista.qranoko.jp/), JSX, SCSS, PostCSS, JS

Реализованные модули на JS:

- BaseComponent (Реализация обновления updateUI через getProxyState)
- InputMask (Добавление шаблона ввода телефонного номера)
- OverlayMenu (Бургер меню)
- Select (Кастомный селект для телефонного input)
- Slider (Слайдер на основе библиотеки Swiper с навигацией вне самого слайдера)
- Tabs (Табуляция с полным функционалом включая управление с клавиатуры и анимацией перемещения заднего фона для
  активного таба)
- VideoPlayer (Базовая настройка видеоплеера)
